Week 2 Key Build — PipelineRunner Class
Design a PipelineRunner class: accepts a list of (name, func) steps, runs them in sequence passing output forward, logs each step with timing using a @timer decorator, handles errors (configurable: halt or skip). Write pytest tests for it.
This is the most important Python exercise of Month 1. Combines OOP, decorators, error handling, and logging. This pattern appears at JPMorgan, Deloitte, and Walmart Python screens.

Week 3 Exercise — The 3 Core Patterns
1) Write a generator that reads a CSV file one row at a time and yields dicts. 2) Write @retry(max_attempts=3, delay=1.0) decorator with exponential backoff. 3) Write a DBConnection context manager that commits on success and rolls back on exception. Write all 3 from memory without looking.
These 3 are the most commonly asked Python patterns in senior DE interviews. If you can write all 3 from memory in under 30 min, you're ready for the Python screen.
