Portfolio Project 3 — RAG Pipeline
Build a RAG system: ingest 10–20 PDF/text documents → chunk by semantic boundaries → generate embeddings → store in FAISS → build a Streamlit query interface with source citation. This is the data engineering pipeline that powers a RAG system — you're not training a model, you're building the data infrastructure.
This is your answer to every "have you worked with AI?" question. Closes the exact gap that cost you the Microsoft interview.
